# kudu

A simple CLI todo list application written in Rust. Data is stored locally in a JSON file.

## Installation

Make sure you have `rustc` and `cargo` in your system.

Clone this project, then install with this command:

```bash
cargo install --path .
```

## Usage

```bash
kudu <COMMAND>
```

\<COMMAND\>:

- add: Add a new todo item
- list: List all todo items
- done: Mark a todo item as completed
- delete: Delete a todo item
